VirtualList usage
The row function describes only the visible content. Virtualization remains the component's job.
scala
val items = new ListProperty[ShowcaseItem]() virtualList(items, estimateHeightPx = 64, crawlable = true) { (item, index) => div { style { height = s"${item.height}px" } text = s"$index - ${item.title}" } }
Crawlable VirtualList
As with the TableView, SSR renders a stable window and links to the next page.
text
/virtual-list?offset=0&limit=50 rendert Items 0 bis 49 /virtual-list?offset=50&limit=50 rendert Items 50 bis 99 Browser: scrollTop startet bei offset * estimateHeight SSR: More-Link zeigt auf offset + limit
Async route usage
The route remains the SSR shell. The VirtualList itself is configured with crawlable = true and reads offset/limit from the route context.
scala
asyncRoute("/virtual-list") { page { VirtualListViewPage.render() } } // In der Page: virtualList(items, estimateHeightPx = 64, crawlable = true) { (item, index) => div { style { height = s"${item.height}px" } text = s"$index - ${item.title}" } }
