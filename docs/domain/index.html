Domain layer
The model is not just data, but metadata with behavior.
JFX2 registers domain classes through reflection descriptors. From that come automatic form bindings, validators from annotations, and JSON mapping that unwraps properties and lists correctly.

Descriptor
Structure stays explicit
Classes, properties, and types remain readable at runtime without string magic in the form.
Annotation
Rules stay on the model
Validation arises from the annotations on the domain properties.
Mapper
JSON stays simple
Property[String] becomes String, ListProperty[Email] becomes Array.
DSL usage
Domain binding is name-based: input(\"email\") binds to user.email.
scala
val user = new User() form(user) { inputContainer("Name") { input("name") {} } subForm("address") { inputContainer("Stadt") { input("city") {} } } } val mapper = new JsonMapper() val json = mapper.serialize(user) val copy = mapper.deserialize[User](json)
