Standalone-Nutzung
Die Property bleibt explizit. Dadurch ist sofort klar, wohin der Wert fließt.
scala
val name = Property("") inputContainer("Enter name...") { input("name", standalone = true) { addDisposable(stringValueProperty.observe(name.set)) } }

Domain-Binding
Durch die Übergabe eines Objekts an form finden Inputs automatisch die passenden Properties.
scala
val user = new User() form(user) { input("name") // Binds to user.name (Property[String]) subForm("address") { input("city") // Binds to user.address.city } }

Form- und DI-Nutzung
Für größere Formulare ist der Form-Kontext die ruhigere Struktur.
scala
import jfx.form.Form.{form, controls} form { inputContainer("Email") { input("email") { validators += EmailValidator() } } button("Submit") { onClick { _ => if (!controls.get.exists(_.validate(true).nonEmpty)) { println("Success!") } } } }
