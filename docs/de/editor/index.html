Lexical as an SSR-safe client island.
Lexical island
The editor renders content immediately and only becomes Lexical in the browser.
JFX2 treats the editor as a ClientSideComponent: SSR gets a stable fallback with the same shell layout, hydration replaces the surface with Lexical, and external property changes flow back into the editor.

Editor showcase: one document with all core content types
